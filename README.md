📊 HR Analytics Dashboard

🚀 Project Overview
-----------------------
This project is an end-to-end HR Analytics solution built using MySQL and Power BI. The objective is to analyze employee data and identify attrition trends, workforce distribution, and salary insights.

🛠 Tools Used
----------------
1. MySQL (Data Modeling & Normalization)
2. Power BI (Data Visualization)
3. DAX (KPI Calculations)
4. Excel (Initial Data Analysis)

📌 Project Workflow
------------------------
1. Data Cleaning and Analysis in Excel
2. Database Normalization in MySQL
3. Created relational tables with Primary & Foreign Keys
4. Built SQL View for analytics-ready dataset
5. Connected MySQL to Power BI
6. Created DAX Measures for KPIs
7. Designed Interactive Dashboard

📊 Key KPIs
--------------
1. Total Employees: 1470
2. Attrition Rate: 16.1%
3. Average Experience: 11 Years
4. Average Age: 37 Years

🔍 Key Insights
--------------------
1. Highest attrition in Research & Development
2. Age group 30–40 shows maximum exits
3. Life Sciences field has highest turnover
